# Animatek WordPress Theme
animatek theme based on tailpress
V.1.0

Tema personalizado para Animatek.net, construido sobre [TailPress](https://tailpress.io) y Tailwind CSS.
## Características
- Diseño moderno y minimalista con Tailwind CSS.
- Integración con Tutor LMS para cursos.
- Páginas personalizadas para landing pages, dashboard de usuario y librería sonora.
- Optimización de rendimiento (Vite para assets).
## Requisitos
- WordPress >= 6.0
- PHP >= 8.0
- Composer
- Node.js & NPM
